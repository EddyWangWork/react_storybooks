# 三国演义 — StoryBooks (Vite + React + Tailwind)

这是一个最小示例项目，演示如何使用 Vite + React + Tailwind 创建一个以《三国演义》为主题的故事展示网站。

## 已包含

- Vite + React (JSX)
- Tailwind CSS 配置 (tailwind.config.cjs, postcss.config.cjs)
- 简单组件：`Hero`, `StoryCard`
- 示例故事数据：`src/data/stories.js`

## 快速开始（Windows, cmd.exe）

1. 在项目目录中安装依赖：

```cmd
cd /d d:\Eddy\Projects\StoryBooks
npm install
```

2. 启动开发服务器：

```cmd
npm run dev
```

3. 在浏览器打开提供的本地地址（通常为 http://localhost:5173 ）

## 可选改进

- 补充更多故事与人物卡片。
- 添加路由（React Router）以实现阅读详细页。
- 增加搜索、过滤和阅读进度保存。
